# Lessons learned

- A publication archive containing every vector figure is not a numerical data
  release. This paper supplies 27 active figure assets but no simulation
  generator or raw samples.
- Panel-level scope matters. A printed analytic fit can be reproduced exactly
  while the finite-size markers in the same panel remain unavailable.
- A transparent proxy is useful for testing a mechanism only when it remains a
  permanent public state rather than being promoted to a surface-code claim.
- More compute cannot recover missing circuit definitions, random seeds, fit
  windows, or decoder revisions. Those are provenance blockers.
- Generated CSV data should precede plotting, and every public figure should
  have a machine-readable check describing its parameters and acceptance rule.
