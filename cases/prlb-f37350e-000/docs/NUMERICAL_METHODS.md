# Numerical methods

All calculations use double precision. Scalar benchmark values are closed-form evaluations. Fig. 1 uses 1,000 uniformly spaced density points over `0.12–0.38 g cm^-3`. Acceptance checks include the exact resonance values `K_±=±1`, `max|K_+K_-+1|`, and exact table-value tolerances. No optimization, stochastic sampling, GPU, or A100 is required.
