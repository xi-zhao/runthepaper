# Lessons learned

## What failed

- A formula true at the simultaneous minimum-resource edge was promoted across a weak-inequality parameter domain.
- Informal “one component contributes” reasoning ignored the particle-number support of a triple annihilator.
- A coherent kick generated by a syndrome projector was treated as nontrivial on a state that the same task declares stabilized.
- A unitary was specified on one code vector and then used to grade an overlap probing an orthogonal error sector.

## New Failure Modes

1. Equality-conditioned boundary defects omitted from a partial-isometry identity.
2. Operator monomial support not checked before fermionic sign bookkeeping.
3. Stabilizer premise contradicted by a later coherent-noise answer.
4. Unitary completion outside the stated subspace silently assumed unique.

## What worked

- Starting from physical Jordan-Wigner operators made every boundary and sign visible.
- Four tiny parameter regimes separated top and bottom edge effects cleanly.
- The source's own strict assumptions immediately distinguished paper claims from benchmark additions.
- Two legal unitary completions gave a constructive, reviewable nonuniqueness proof.
