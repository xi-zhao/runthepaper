# Similarity scorecard

All four benchmark targets receive full feature, numeric, and contracted-scope
credit. Relative differences range from about 0 ppm to 6.34 ppm, compatible
with the different print precisions in the frozen answers. The normalized
machine-readable score is 100/100 at `numerical_feature_reproduction`: the
benchmark parameters are `not_applicable` to source-paper panels and every
rendered benchmark artifact is `exploratory` relative to the paper. The
machine-readable scorecard is `outputs/checks/similarity_scorecard.json`.

Paper-wide visual similarity is not scored as complete: benchmark points are
not a substitute for any complete PEPS figure.
