# Derivation

The full source acceptance criterion contains the reactive endpoint indicator, the product excluding internal basin hits, and

$$
\min\left(1,\frac{\sum_{i\in X}e^{-\beta U_{bias}(x_i)}}{\sum_{i\in X'}e^{-\beta U_{bias}(x'_i)}}\right).
$$

For Fig. 2B we evaluate

$$
A(x^{(0)})=-\log\int dx^{(1)}e^{-U(x^{(0)},x^{(1)})},
$$

using the SI potential `U=alpha/4*((r²-4)²+(x1)²)` and shift each curve to zero minimum.
